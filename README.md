# QA Automation - Cypress

E2E test automation framework using Cypress for web platform testing.

## 🛠️ Tech Stack
- Cypress v15
- JavaScript
- Node.js

## 📋 Test Cases
- ✅ Login exitoso con credenciales válidas
- ✅ Login fallido con contraseña incorrecta  
- ✅ Login fallido con usuario bloqueado

## 🚀 How to Run
```bash
npm install
npx cypress open
```
